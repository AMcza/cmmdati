12. 实现一个简单的Python函数，该函数接受一个整数列表并返回其中的最大值。

def find_max(numbers):
    max_value = numbers[0]
    for number in numbers:
        if number > max_value:
            max_value = number
    return max_value

# 示例调用
numbers = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5]
print(find_max(numbers))  # 输出应为 ____
