Python的基本数据类型包括整数（int）、浮点数（float）、布尔值（bool）和字符串（str）。这些数据类型是不可变的，意味着一旦创建就不能更改其内容。

整数（int）是用于表示整数值的数据类型。例如： 

x = 10
y = -5
z = 123456789012345678901234567890
Python中的整数可以非常大，不受固定位数的限制。

浮点数（float）用于表示小数值。例如： 

a = 3.14
b = -0.001
c = 1.2e10
浮点数在计算机中以二进制形式存储，可能会有一些精度误差。

布尔值（bool）只有两个可能的值：True和False。它们通常用于条件判断。例如： 

is_valid = True
has_error = False
if is_valid:
    print("The operation is valid.")
else:
    print("The operation has an error.")

字符串（str）用于表示文本数据。字符串可以用单引号、双引号或三引号括起来。例如： 

greeting = 'Hello, World!'
message = "Welcome to Python."
multiline_text = """This is a string that spans
multiple lines."""
字符串是不可变的，但可以通过切片等方式获取子字符串。

此外，Python还提供了其他一些重要的内置数据类型，如列表（list）、元组（tuple）、集合（set）和字典（dict）。这些数据类型是可变的，并且可以包含多种不同类型的数据。例如： 

my_list = [1, 2, 3, 4, 5]
my_tuple = (1, 2, 3, 4, 5)
my_set = {1, 2, 3, 4, 5}
my_dict = {'a': 1, 'b': 2, 'c': 3}
列表是最常用的数据结构之一，支持动态增删元素；元组与列表类似，但不可变；集合用于存储唯一的元素；字典则用于存储键值对。
